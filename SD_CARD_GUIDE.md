# Руководство по работе с SD-картой (Waveshare ESP32-S3-Touch-LCD-7)

В данном устройстве кардридер подключен по интерфейсу **SPI**. Особенностью является то, что пин выбора чипа (CS) управляется не напрямую процессором, а через встроенный расширитель портов **CH32V003**.

## 🛠 Техническое подключение (Pinout)

| Сигнал | Пин ESP32-S3 | Примечание |
| :--- | :--- | :--- |
| **CLK** | GPIO12 | SPI Clock |
| **MOSI** | GPIO11 | SPI Data Out |
| **MISO** | GPIO13 | SPI Data In |
| **CS** | **Expander Pin 4** | Управляется через `waveshare_io_hub` |

---

## ⚙️ Конфигурация ESPHome (`panel-s3.yaml`)

Для работы карты необходимо определить шину SPI и компонент `sdcard`:

```yaml
spi:
  clk_pin: GPIO12
  mosi_pin: GPIO11
  miso_pin: GPIO13

sdcard:
  cs_pin:
    waveshare_io_ch32v003: waveshare_io_hub
    number: 4
```

---

## 🎨 Настройка графики (`gui.yaml`)

Чтобы библиотека LVGL могла открывать файлы с карты, нужно зарегистрировать файловую систему и декодеры:

```yaml
lvgl:
  filesystem:
    - type: SDCARD
      letter: S # Карта будет доступна как S:/
      
  decoders:
    - type: PNG
    - type: JPG
```

## 🖼 Использование фоновых изображений

Для автоматической смены фона при переключении страниц используйте триггер `on_show` в YAML-файле соответствующей страницы:

### Пример для страницы `page_dash.yaml`:
```yaml
id: page_dash
on_show:
  then:
    - lambda: |-
        // Установка фонового изображения из файла на SD-карте
        // Файл должен лежать по пути: backgrounds/page_dash.png
        lv_obj_set_style_bg_img_src(id(page_dash).get_obj(), "S:/backgrounds/page_dash.png", 0);
```

## 📝 Важные примечания и советы

1.  **Форматирование**: Используйте только **FAT32**. Карты с exFAT или NTFS не будут распознаны.
2.  **Формат файлов**: 
    *   **PNG/JPG**: Удобно, но медленно (декодирование 1024x600 занимает ~1-2 сек).
    *   **LVGL Binary (.bin)**: Максимальная скорость. Сконвертируйте картинку через [официальный конвертер LVGL](https://lvgl.io/tools/imageconverter), выбрав формат "Binary".
3.  **Пути к файлам**: Все пути в коде должны начинаться с префикса `S:/`. Например: `S:/images/logo.png`.
4.  **Память**: Декодирование полноэкранного изображения требует около 2.5 MB свободной PSRAM. Устройство имеет 8 MB, чего достаточно для одновременной работы нескольких фонов.
5.  **Питание**: Работа с SD-картой увеличивает пиковое потребление тока. Убедитесь, что блок питания выдает минимум 2А.
